# 🌿 Nisakorn Garden E-Commerce Platform

E-commerce platform demo สำหรับธุรกิจผลไม้สดคัดสรร

## ✨ Features

### 🎭 Multi-Role System
- **👥 B2C Customer** - ลูกค้าทั่วไป (ซื้อปลีก)
- **🏢 B2B Customer** - ลูกค้าธุรกิจ (ขายส่ง)
- **👨‍💼 Staff** - พนักงานคลังสินค้า
- **👨‍💻 Admin** - ผู้ดูแลระบบ

### 🛒 Customer Features
- Shopping cart & Checkout
- Product catalog & Search
- User authentication
- Order history
- Wishlist management

### 💼 Business Features
- B2B wholesale pricing
- Quotation system  
- Inventory management
- Staff operations dashboard
- Admin analytics

## 📋 How to Use

1. Open `index.html` to see the role selector
2. Click on any role card to explore that user experience
3. Navigate through different pages using the navigation menu
4. All pages are fully interactive with mock data
